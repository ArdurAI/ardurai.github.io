How It Works

Governance That
Actually Works

1

Declare Mission

Define scope, tools, resources, and budgets in a cryptographically signed mission document.

mission {
  scope: "code-generation"
  tools: ["file_write", "shell_read"]
  budget: { tokens: 100000 }
}
2

Runtime Enforcement

Every tool call flows through the governance proxy. Policies are checked before execution.

~4ms overhead per call
